<p>A specialisation of the Observations and Measurements NamedValue class that is specific to Metafor.  It contains a constraint on the parameters that may be used. M4A_CompositeValue.parameter is of type NamedValue, to allow a wider range of parameters to be supplied, but M4A_NamedValue can be used to describe Metafor specific parameters.<br /></p>
