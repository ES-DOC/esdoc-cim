<p>Any class or feature with the <<reference>> stereotype uses the attributes of this class.  <br /></p><p><br /></p><p>With all the different ways of pinpointing an XML item, a reference can either use XPATH to directly locate the item or it can just identifiy the document and then use the other attributes (name,type,etc.) to narrow down the particular element within that document.  <br /></p>
