<p>An ensemble is made up of two or more simulations which are to be compared against each other to create ensemble statistics. Ensemble members can differ in terms of initial conditions, physical parameterisation and the model used.  <br /></p><p><br /></p><p>An ensemble bundles together sets of ensembleMembers, all of which reference the same Simulation(Run) and include one or more changes.<br /></p>
