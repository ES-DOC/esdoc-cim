<p>The source/target of a connetion.  This is a DataSource (a ComponnetProperty or DataContent).  This is a separate class in order to associate an instanceID with the DataSource; this is used to identify which particular instance is being coupled in case the same DataSource is used more than once in a coupled model (this may be required for BFG).  Realistically, tihe instanceID is unlikely to be used for a connection, only for a coupling.  It is provided here for consistency.<br /></p>
